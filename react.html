Context and Purpose

This is a small experimental web project based on react and bootstrap. This web page tries to mimic a basic shopping software: users can browse and buy the items they want by filtering and sorting them, and they can also remove items from their shopping cart.

Process

Hi-fi design

The functions

In this bakery store website, the user can choose to use the filter/sort at the left side.

  • Filter by bakery types
  • Filter by bakery dietary restriction
  • Sort by default setting (popularity)
  • Sory by price(low - high) or (high - low)

The users can also add/delete/reset the items in their cart.

  • PROBLEM

    • Creating a static web page is not so hard, but interacting with users and implementing some features requires additional learning. This project aims to solve some functional problems.

  • GOAL

    • This project is a small experiment for me to try REACT and bootstrap for the first time.

  • DISCUSSION

    • Although I didn't spend much time on making the page more responsive based on the project's goals (getting familiar with and using react and bootstrap), I think there's still a lot of room for improvement. For example, on my monitor (24"), the menu is a bit too loose between each item (I designed it with 2 columns), which may distract the user's attention and is not conducive to uniformity of eye movement. Also, I didn't have time to create the feedback part of the "check out" button because of time constraints. Hopefully I would continue working on that!
